Histórico das Matrizes

Conheça a fascinante jornada do desenvolvimento das matrizes na matemática

Século XVIII - Origens

O conceito de matrizes surgiu do estudo de sistemas de equações lineares. Matemáticos como Leibniz e Cramer desenvolveram métodos para resolver estes sistemas.

1855 - Arthur Cayley

Arthur Cayley publicou trabalhos fundamentais sobre a teoria das matrizes, estabelecendo as operações básicas e propriedades algébricas que usamos até hoje.

1850s-1860s - James Joseph Sylvester

Sylvester contribuiu significativamente com a notação de matrizes e o desenvolvimento de invariantes matriciais, consolidando a teoria moderna.

Importância Contemporânea

Matrizes são fundamentais em matemática, estatística, física, engenharia, computação gráfica, machine learning e praticamente todas as ciências aplicadas.

Matrizes na Linguagem R

Compreenda a estrutura e funcionalidade das matrizes em R

O que é uma Matriz em R?

Uma matriz em R é uma estrutura de dados bidimensional (linhas e colunas) que armazena elementos do mesmo tipo. Diferente de um data frame, todos os elementos devem ser do mesmo tipo de dado (numérico, caractere, lógico, etc.).

Estrutura Bidimensional

Uma matriz é organizada em linhas (horizontais) e colunas (verticais). Cada elemento é identificado por sua posição [linha, coluna].

Matriz[i, j] onde i = linha, j = coluna

Função matrix()

A função principal para criar matrizes em R é matrix()

Argumentos principais:

  • data - vetor com os elementos
  • nrow - número de linhas
  • ncol - número de colunas
  • byrow - TRUE para preencher por linha

Exemplo Básico

A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)
A

#      [,1] [,2]
# [1,]    1    3
# [2,]    2    4

Operações Principais

+

Soma: A + B

Soma elemento a elemento

×

Multiplicação: A %*% B

Multiplicação matricial

T

Transposta: t(A)

Troca linhas por colunas

|

Determinante: det(A)

Valor escalar da matriz

⁻¹

Inversa: solve(A)

Matriz inversa (se existir)

Dica: Use dim(A) para verificar as dimensões, nrow(A) para linhas e ncol(A) para colunas.

Exemplo Completo

# Criar duas matrizes
A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)
B <- matrix(c(5, 6, 7, 8), nrow = 2, ncol = 2)

# Operações
soma <- A + B           # Soma elemento a elemento
mult <- A %*% B         # Multiplicação matricial
transp <- t(A)          # Transposta
det_A <- det(A)         # Determinante
inv_A <- solve(A)       # Inversa

# Visualizar resultados
print("Soma:")
print(soma)
print("Multiplicação Matricial:")
print(mult)

Exemplos Práticos em R

Explore diversos exemplos de criação, manipulação e operações com matrizes

Criação Básica de Matriz

Criar uma matriz simples com 2 linhas e 2 colunas usando a função matrix() 

# Criar uma matriz 2x2
A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)
A

# Resultado:
#      [,1] [,2]
# [1,]    1    3
# [2,]    2    4

Preenchimento por Linha

Usar o argumento byrow = TRUE para preencher a matriz por linha em vez de coluna 

# Preencher por linha
B <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3, byrow = TRUE)
B

# Resultado:
#      [,1] [,2] [,3]
# [1,]    1    2    3
# [2,]    4    5    6

Acesso a Elementos

Acessar elementos específicos de uma matriz usando índices [linha, coluna] 

# Acessar elemento na linha 1, coluna 2
A[1, 2]  # Retorna: 3

# Acessar toda a primeira linha
A[1, ]   # Retorna: 1 3

# Acessar toda a segunda coluna
A[, 2]   # Retorna: 3 4

Soma de Matrizes

Somar duas matrizes de mesma dimensão elemento a elemento 

A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)
B <- matrix(c(5, 6, 7, 8), nrow = 2, ncol = 2)

# Soma
C <- A + B
C

# Resultado:
#      [,1] [,2]
# [1,]    6   10
# [2,]    8   12

Multiplicação Matricial

Multiplicar duas matrizes usando o operador %*% 

A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)
B <- matrix(c(5, 6, 7, 8), nrow = 2, ncol = 2)

# Multiplicação matricial
C <- A %*% B
C

# Resultado:
#      [,1] [,2]
# [1,]   19   22
# [2,]   43   50

Transposta de Matriz

Transpor uma matriz (trocar linhas por colunas) usando t() 

A <- matrix(c(1, 2, 3, 4, 5, 6), nrow = 2, ncol = 3)
A

# Transposta
At <- t(A)
At

# Resultado:
#      [,1] [,2]
# [1,]    1    4
# [2]    2    5
# [3,]    3    6

Determinante de Matriz

Calcular o determinante de uma matriz quadrada usando det() 

A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)

# Determinante
d <- det(A)
d

# Resultado: -2
# Cálculo: (1 × 4) - (2 × 3) = 4 - 6 = -2

Inversa de Matriz

Calcular a inversa de uma matriz quadrada invertível usando solve() 

A <- matrix(c(1, 2, 3, 4), nrow = 2, ncol = 2)

# Inversa
Ainv <- solve(A)
Ainv

# Verificação: A %*% Ainv = I (matriz identidade)
A %*% Ainv

Aplicação Real: Cálculo de Faturamento

Veja como matrizes resolvem problemas práticos em negócios

Cenário

Uma empresa vende 3 produtos diferentes em 2 meses. Queremos calcular o faturamento total de cada produto usando multiplicação matricial.

Dados:

  • Produto 1: 100 unidades (mês 1), 150 unidades (mês 2)
  • Produto 2: 80 unidades (mês 1), 120 unidades (mês 2)
  • Produto 3: 50 unidades (mês 1), 90 unidades (mês 2)
  • Preços: R$ 10, R$ 15, R$ 20

Solução Passo a Passo

1. Criar matriz de quantidades (3×2)

Cada linha = um produto, cada coluna = um mês

2. Criar matriz de preços (1×3)

Uma linha com os preços dos 3 produtos

3. Multiplicação matricial

preços %*% quantidade = faturamento por mês

4. Interpretar resultados

Cada coluna mostra o faturamento total do mês

Código R Completo

# Matriz de quantidades (3 produtos × 2 meses)
quantidade <- matrix(
  c(100, 150,
    80,  120,
    50,  90),
  nrow = 3,
  byrow = TRUE
)

# Matriz de preços (1 × 3)
precos <- matrix(c(10, 15, 20), nrow = 1)

# Multiplicação matricial
# Resultado: 1×2 (faturamento por mês)
faturamento <- precos %*% quantidade

# Visualizar resultado
print("Faturamento por mês:")
print(faturamento)

# Resultado esperado:
# [,1]  [,2]
# 3850  5850

# Interpretação:
# Mês 1: R$ 3.850
# Mês 2: R$ 5.850
